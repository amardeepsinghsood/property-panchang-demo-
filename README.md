# Property Panchang - Website Repository

This repository contains the full source code for the **Property Panchang** premium real estate website, optimized for **GitHub Pages**.

## 🚀 How to Deploy to GitHub

Since this project uses a **flat-file structure** (all images and code in the root), it is extremely easy to upload and serve via the GitHub web interface.

### Step 1: Create a New Repository
1. Log in to your GitHub account.
2. Click the **+** icon in the top right and select **New repository**.
3. Name your repository (e.g., `property-panchang`).
4. Set it to **Public** and click **Create repository**.

### Step 2: Upload Files
1. In your new repository, click the **Add file** button and select **Upload files**.
2. Drag and drop **all files** from your local `property panchang web start` folder directly into the upload area.
3. Once uploaded, click **Commit changes** at the bottom.

### Step 3: Enable GitHub Pages
1. Go to the **Settings** tab of your repository.
2. Select **Pages** from the left sidebar.
3. Under **Build and deployment** > **Branch**, select `main` (or `master`) and click **Save**.
4. Your site will be live at: `https://your-username.github.io/repository-name/`

## 📁 File Structure
- `index.html`: Main landing page
- `*.html`: Project detail pages (e.g., `trump-tower.html`)
- `style.css`: Main stylesheet
- `script.js`: Interactive elements & animations
- `*.jpg` / `*.svg`: Images and branding assets (directly in root for ease of upload)
